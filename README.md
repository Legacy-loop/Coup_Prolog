#Problema das 8 Rainhas em Prolog
Descrição
Este projeto implementa o Problema das 8 Rainhas utilizando Prolog.

O objetivo é posicionar 8 rainhas em um tabuleiro 8x8 sem que nenhuma ataque outra.

O sistema utiliza:

fatos;

regras;

inferência lógica;

backtracking.

Funcionalidades
Inserção manual de rainhas;

Verificação de conflitos;

Resolução automática;

Visualização do tabuleiro.

Como Executar
Carregar arquivo
consult('arquivo.pl').
Iniciar sistema
?- iniciar.
Comandos
Inserir rainha
(3,5).
Listar rainhas
lista.
Resolver automaticamente
resolver.
Encerrar
sair.
Consultas Lógicas
Verificar conflito
?- conflito(1,1,3,3).
Verificar posição segura
?- segura(4,5).
Conceitos Utilizados
Programação lógica;

Regras;

Inferência lógica;

Backtracking;

Paradigma declarativo.
